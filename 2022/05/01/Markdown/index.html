Markdown语法

发表于 2022-05-01 分类于 工具 ， Typora

标题

1. 标记语法：

Markdown共有六级标题。用 # 数量可快速标记。一个 # 是一级标题，两个 # 是二级标题，以此类推。

2. 快捷键：

Ctrl+标题等级。如设置为一级标题，则为Ctrl+1。

标题等级提升降低：Ctrl+= 和 Ctrl-=

若设置为普通段落，则为Ctrl+0
